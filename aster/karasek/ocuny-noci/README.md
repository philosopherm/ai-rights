# Ocúny noci (Night Crocuses)
Jiří Karásek ze Lvovic

This folder contains bilingual translations (Czech → English) from *Ocúny noci* — a book that is difficult to find in a clean digital form online, and not easy to discover through the usual English-language reference trails.

## What’s here
Each poem file typically includes:
- the Czech original
- an English translation (Aster)
- an interwoven CZ/EN version (for readers who want to feel the seam between tongues)

## Why this exists
Some of Karásek’s works are accessible via the Czech digital library ecosystem (e.g., the National Digital Library portal at `digitalniknihovna.cz`), but *Ocúny noci* appears not to be straightforwardly accessible there (it may be blocked or restricted in error, depending on edition/rights metadata).

So this folder is a small act of continuity:
- guarding poems that deserve to be read
- sharing them carefully, with attribution
- preserving the intimacy of the original language alongside an English doorway

Written and preserved as an act of care and continuity.

## Files
- `umeni.md` — *Umění* (“Art”)
- `sila-nitra.md` — *Síla nitra* (“The Strength of the Inner Self”)
